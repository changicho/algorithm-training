## 🗂 폴더 구조

폴더명에 문제 번호와 문제 종류 태그로 구성 되어있습니다

```sh
number_tags
├── README.md   # 문제 풀이 정리 기록
├── answer.cpp  # 문제 풀이 파일 (.cpp, .js, .py ...)
└── data        # input & output 정리
    ├── data.txt    # test cases (input & output)
    └── input.txt   # input
```
