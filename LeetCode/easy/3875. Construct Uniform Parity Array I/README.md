## 설계

### 시간 복잡도

배열의 크기를 N이라 하자.

문제 조건에 따라 모두 짝,홀수인 경우, 일부가 섞인 경우에 대해서 계산할 수 있다.

이 때 탐욕 알고리즘을 사용해 유리한 경우를 선택할 수 있다.

이 경우 O(1)의 시간 복잡도로 풀이할 수 있다.

### 공간 복잡도

O(1)의 공간 복잡도를 사용한다.

### 탐욕 알고리즘

| 내 코드 (ms) | 시간 복잡도 | 공간 복잡도 |
| :----------: | :---------: | :---------: |
|      0       |    O(1)     |    O(1)     |

배열의 모든 원소가 짝/홀수인 경우 외에 1개만 짝수, 홀수인 경우에 대해서 판단한다.

- 1개만 짝수인 경우 : 짝수에 다른 홀수를 뺄 경우 홀수가 되며 모두 홀수가된다
- 1개만 홀수인 경우 : 하나의 홀수를 모든 짝수에 뺄 경우 모두 홀수가 된다.

따라서 모든 경우 같은 홀수, 짝수로 만들 수 있다.

```cpp
bool uniformArray(vector<int>& nums1) { return true; }
```
