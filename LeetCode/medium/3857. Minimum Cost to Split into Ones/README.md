## 설계

### 시간 복잡도

입력받은 수를 N이라 하자.

조건을 만족하기 위해선 1씩 감소시키며 쪼개야 한다. 이를 직접 구현하면 O(N)이다.

수학을 사용할 경우 O(1)의 시간 복잡도로 풀이할 수 있다.

### 공간 복잡도

정답에 O(1)의 공간 복잡도를 사용한다.

### 수학

| 내 코드 (ms) | 시간 복잡도 | 공간 복잡도 |
| :----------: | :---------: | :---------: |
|      0       |    O(1)     |    O(1)     |

특정 수 X = A + B로 쪼개며 비용은 A \* B라 하자.

이 경우 비용을 최소화 하기 위해서는 X = 1 + (X - 1)로 쪼개야한다.

이를 N이 1이 될 때까지 반복할 경우 (N - 1) + (N - 2) + ... + 1 = N \* (N - 1) / 2의 비용이 든다.

```cpp
int minCost(int n) { return (n) * (n - 1) / 2; }
```
