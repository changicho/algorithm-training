# A - Not Found

|  100   |

## 설계

### 시간 복잡도

입력받은 문자열의 길이를 N이라 하자.

알파벳 소문자를 모두 완전탐색할 경우 O(N)의 시간 복잡도를 사용한다.

### 공간 복잡도

완전 탐색에 별도의 공간을 사용하지 않으므로 O(1)의 공간 복잡도를 사용한다.

### 완전 탐색

| 내 코드 (ms) | 시간 복잡도 | 공간 복잡도 |
| :----------: | :---------: | :---------: |
|      2       |    O(N)     |    O(1)     |

각 알파벳 소문자별로 문자열에 존재하는지 모두 탐색한다.

```cpp
char solution(string s) {
  for (char c = 'a'; c <= 'z'; c++) {
    if (s.find(c) == string::npos) {
      return c;
    }
  }

  return 'a';
}
```
