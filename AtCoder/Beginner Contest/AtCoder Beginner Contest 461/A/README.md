## 설계

### 시간 복잡도

입력받은 수를 비교하는데 O(1)의 시간 복잡도를 사용한다.

### 공간 복잡도

비교에 O(1)의 공간 복잡도를 사용한다.

### 비교

| 내 코드 (ms) | 시간 복잡도 | 공간 복잡도 |
| :----------: | :---------: | :---------: |
|      1       |    O(1)     |    O(1)     |

```cpp
bool solution(int a, int d) { return a <= d; }
```
