## 설계

### 시간 복잡도

숫자 세 자리를 비교하는데 O(1)의 시간 복잡도를 사용한다.

### 공간 복잡도

숫자 비교에 O(1)의 공간 복잡도를 사용한다.

### 비교

| 내 코드 (ms) | 시간 복잡도 | 공간 복잡도 |
| :----------: | :---------: | :---------: |
|      1       |    O(1)     |    O(1)     |

숫자를 문자열로 변환해 각 자리수가 모두 같은지 비교한다.

```cpp
bool solution(int n) {
  string s = to_string(n);

  return s[0] == s[1] && s[1] == s[2];
}
```
