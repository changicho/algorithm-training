## 설계

### 시간 복잡도

문자열의 길이를 N이라 하자.

N이 5로 나누어 떨어지는 지 확인하는데 O(1)의 시간 복잡도를 사용한다.

### 공간 복잡도

모듈러 연산에 O(1)의 공간 복잡도를 사용한다.

### 나머지

| 내 코드 (ms) | 시간 복잡도 | 공간 복잡도 |
| :----------: | :---------: | :---------: |
|      2       |    O(1)     |    O(1)     |

s의 길이는 최대 10이며 5의 배수여야만 "YES"를 출력한다.

```cpp
bool solution(string& s) {
  int size = s.size();

  return size % 5 == 0;
}
```
