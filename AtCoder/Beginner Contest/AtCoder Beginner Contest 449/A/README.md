## 설계

### 시간 복잡도

입력받은 수를 N이라 하자.

N이 지름인 원의 넓이를 반환한다. 이에 O(1)의 시간 복잡도를 가진다.

### 공간 복잡도

계산에 O(1)의 공간 복잡도를 가진다.

### 수학

| 내 코드 (ms) | 시간 복잡도 | 공간 복잡도 |
| :----------: | :---------: | :---------: |
|      1       |    O(1)     |    O(1)     |

```cpp
double solution(int n) {
  double pi = 3.141592653589793;
  double r = (double)n / 2;
  return pi * r * r;
}
```
