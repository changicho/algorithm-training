## 설계

### 시간 복잡도

입력받은 두 수를 더하고 제곱하는데 O(1)의 시간 복잡도가 필요하다.

### 공간 복잡도

제곱에 O(1)의 공간 복잡도가 필요하다.

제곱수는 int를 벗어나지 않는다.

### 정리

| 내 코드 (ms) | 시간 복잡도 | 공간 복잡도 |
| :----------: | :---------: | :---------: |
|      1       |    O(1)     |    O(1)     |

```cpp
int solution(int a, int b) { return (a + b) * (a + b); }
```
