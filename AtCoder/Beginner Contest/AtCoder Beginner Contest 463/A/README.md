## 설계

### 시간 복잡도

입력받은 두 수를 계산하는데 O(1)의 시간 복잡도를 사용한다.

### 공간 복잡도

두 수를 계산하는데 O(1)의 공간 복잡도를 사용한다.

### 수학

| 내 코드 (ms) | 시간 복잡도 | 공간 복잡도 |
| :----------: | :---------: | :---------: |
|      1       |    O(1)     |    O(1)     |

비율을 계산한다.

bool solution(int x, int y) { return x * 9 == y * 16; }
